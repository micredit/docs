## 创建资产钱包

进入顶部`Web3`，进入Web3钱包。首次使用，选择`创建钱包`。

点击`立即创建`按钮

<img src="images/okx-wallet-guide/02-okx-create-wallet.jpg" width="400" />


根据提示，设置钱包的密码，完成后点击`继续`。

<img src="images/okx-wallet-guide/03-okx-setup-wallet-password.jpg" width="400" />

钱包创建完成，根据提示，开始备份钱包，默认选择`手动备份`。

<img src="images/okx-wallet-guide/04-okx-backup-wallet.jpg" width="400" />

认真看完文字提示，点击`继续`按钮。

<img src="images/okx-wallet-guide/05-okx-backup-wallet-why.jpg" width="400" />

认真看完文字提示，点击`开始`按钮。

<img src="images/okx-wallet-guide/06-okx-backup-wallet-start.jpg" width="400" />

根据提示，输入一次钱包密码，点击`确认`按钮。

<img src="images/okx-wallet-guide/07-okx-backup-wallet-password.jpg" width="400" />

再次阅读屏幕风险提示，点击`我知道了`按钮。

<img src="images/okx-wallet-guide/08-okx-backup-wallet-mention.jpg" width="400" />

抄写，备份提示词。为了您的财产安全，请勿截图，拍照，存手机。强力建议用纸币记录下来，并收藏保管好。

<img src="images/okx-wallet-guide/09-okx-backup-wallet-phrases.jpg" width="400" />

点击`继续`按钮，进入钱包首页。

<img src="images/okx-wallet-guide/10-okx-wallet-home.jpg" width="400" />


## 添加$MICT代币Token

点击钱包币种右侧的功能按钮。

<img src="images/okx-wallet-guide/10-okx-wallet-home.jpg" width="400" />

点击"自定义币种"。

选择网络：TON。 Jetton Master输入$MICT代币的合约地址：`EQDTtusdnxjU7O5qyZyPMNBhxb96cZ69_5x17KaovygF2KLr`。确认后代币名称和精准度会自动补全。

<img src="images/okx-wallet-guide/12-okx-wallet-add-mict.jpg" width="400" />

点击`确认`按钮后，就可以看到`$MICT`代币了。

<img src="images/okx-wallet-guide/13-okx-walle-added-token.jpg" width="400" />

点击返回按钮，回到钱包首页，可有看到MICT的名称和LOGO。

## 接收$MICT并交易

<img src="images/okx-wallet-guide/14-okx-wallet-home-mict.jpg" width="400" />

点击MICT代币名称，可进入MICT资产收发界面。

<img src="images/okx-wallet-guide/15-okx-mict-home.jpg" width="400" />

点击`接收`按钮，可以向他人展示自己的钱包地址以及地址二维码。以后，别人让你发钱包地址，就发这里的地址。

<img src="images/okx-wallet-guide/16-okx-mict-receive.jpg" width="400" />

如果有人向你转账`$MICT`，在钱包余额中能看到。

<img src="images/okx-wallet-guide/17-okx-mict-balance.jpg" width="400" />

点击链上行情，可以查看币价，并进行交易。推荐点击收藏按钮，方便后续使用。

<img src="images/okx-wallet-guide/18-okx-micto-trading.jpg" width="400" />

由于MICT是在TON网络上发行的代币，因此买卖MICT之前，您的钱包需要拥有一定的TON作为交易手续费。

### Edit and preview

  <Accordion icon="rectangle-terminal" title="Preview changes">
    Previewing helps you make sure your changes look as intended. We built a
    command line interface to render these changes locally. 
    1. Install the
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) to preview the
    documentation changes locally with this command: ``` npm i -g mintlify ```
    2. Run the following command at the root of your documentation (where
    `docs.json` is): ``` mintlify dev ```
    <Note>
      If you’re currently using the legacy ```mint.json``` configuration file, please update the Mintlify CLI:


      ```npm i -g mintlify@latest```
      And run the new upgrade command in your docs repository:

      ```mintlify upgrade```
      You should now be using the new ```docs.json``` configuration file. Feel free to delete the ```mint.json``` file from your repository.
    </Note>
  </Accordion>
</AccordionGroup>
